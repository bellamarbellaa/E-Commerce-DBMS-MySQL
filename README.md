# E-Commerce-DBMS-MySQL
A simple A MySQL-based e-commerce database system designed to manage customers, shops, products, transactions, reviews, and shopping carts. This project demonstrates relational database design and SQL implementation through business-oriented scenarios involving inventory management, customer analytics, sales tracking, and transaction processing.

Files in this repository include: 
1. Data Definition & Manipulation Queries - Contains database creation, schema design, constraints, foreign key relationships, and sample data population using DDL and DML statements.
2. Data Query Language Questions & Answers - Contains SQL queries and solutions covering data retrieval, filtering, sorting, joins, aggregations, subqueries, views, and stored procedures used to answer business-related questions.
